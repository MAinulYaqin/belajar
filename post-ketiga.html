Cara instalasi file .deb menggunakan terminal pada linux
LINUX / TIPS & TRICK / TUTORIAL

pernah gk nginstall libreOffice ? kalau pernah tau kan file .deb yang buanyak waktu di ekstrak file libreOfficenya.. pasti mikir, waduh.. kalau di install satu2 bakal makan waktu banyak nih. nah.. cara cepat untuk nginstall file .deb yang buanyak itu make dpkg gan/sist.. cara nya gini. cekidot

caranya mudah, pertama adalah cd atau change directory ke tempat file itu berada.

$ cd ~/Nama directory

kalau gk mau repot, langsung aja klik kanan di directory-nya lalu klik Open In Terminal. lalu ketik command :

$ sudo dpkg -i NamaDirectory.deb

kalau single atau cuma satu file ditulis gini :

$ sudo dpkg -i NamaFile.deb

Cara unnistall.

kalau ada cara install pasti ada cara unnistall - nya.. oke, kalau mau unnistall aplikasi tersebut, ketik commang seperti ini :

$dpkg -r atau --remove atau -p atau --purge [nama paket aplikasi]

Keterangan : -r dan --remove menghapus paket aplikasi akan tetapi tidak membersihkan file konfigurasi. -p dan --purge menghapus seluruh paket aplikasi serta file konfigurasi bawaan (cache, profile, temporary data,etc).

Semoga bermanfaat gan sist.
