Visual Studio 2017 telah update dengan fitur barunya
INFO / PEMROGRAMAN / WINDOWS

Microsoft telah mengumumkan kemampuan terbaru secara umum dari Visual Studio 2017 yang baru saja dirilisnya. Peluncuran ini juga menandakan usia Visual Studio yang sudah 20 tahun di mana mencakup environment desktop C++, Visual Basic dan Java dialectJ ++. Visual Studio 2017 adalah versi ke-11 dari IDE yang populer ini.

Para developer bisa menggunakan Visual Studio 2017 untuk membuat aplikasi untuk Android, iOS, Windows, Web dan Cloud. "membuat code dengan cepat, debug dan diagnose dengan mudah, test dan rilis dengan percaya diri," tambah Microsoft.

#Visual Studio 2017 - Produktifitas dan Performa, Mobile, Cloud dan DevOps

Pada rilisnya kali ini, Microsoft telah fokus pada 4 faktor : produktifitas dan performa, mobile, cloud dan DevOps. Dalam catatan rilisnya Microsoft mengatakan VS 2017 ini dirilis dengan membawa kinerja yang ringan dan pengalaman instalasi dengan rujukan modul. VS 2017 ini memungkinkan developer hanya mendapatkan komponen yang mereka butuhkan.

Microsoft juga mengklaim Visual Studio 2017 bekerja dengan 3 kali lebih cepat dibanding dengan versi Visual Studio 2015. Peluncuran VS 2017 ini didesain untuk membantu team bekerja dengan lebih baik. Dengan membawa tool yang menyediakan pengembangan NET Core 1.0 dan 1.1 app, microservice, dan kontainer docker.

Integrasi Xamarin ke dalam Visual Studio 2017 telah dibuat dengan lebih baik dengan hadirnya Xamarin Forms previewer dan peningkatan pada Xamarin Forms XAML. Tidak hanya itu peningkatan pada integrasi Git pun akan menjadi sesuatu yang disukai oleh DevOps. Visual Studio kini membuat penggunaan Git dapat dieksekusi. Secara keseluruhan, lebih banyak lagi fungsi Git yang bisa digunakan secara langsung dalam IDE.

Untuk informasi lebih detail tentang rilisnya ini, silahkan untuk membacanya langsung pada catatan perilisannya.

Selain dengan rilisnya Microsoft Visual Studio 2017, terdapat juga update untuk Preview 4 Visual Studio untuk Mac dan Team Foundation Server 2017.
