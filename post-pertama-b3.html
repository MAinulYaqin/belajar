Apa sih virtual-box itu ?
INFO

Mungkin bagi agan sist yang dulunya anak IT nama Virtualbox sudah tidak asing lagi didengar bukan ?. Biasanya Virtualbox digunakan untuk mencoba sebuah sistem operasi tanpa harus menimpa sistem operasi yang sudah ada. Nah tahukah agan/sist jika fungsinya tidak terbatas hanya di situ saja ? kali ini akan kita bahas fungsi-fungsi Virtualbox.

#Pengertian

Virtualbox adalah software gratis milik Oracle yang fungsi utamanya adalah mem-visualisasi-kan sebuah atau banyak Sistem Operasi (OS) di dalam Sistem Operasi utama kita.

Jadi sobat tidak perlu punya banyak komputer untuk memakai atau menguji beberapa Sistem Operasi. Misalnya: OS utama agan/sist adalah Windows 8 namun juga ingin merasakan OS lain yang di perkenalkan teman, ‘Linux Mint’. Maka bisa menggunakan Virtualbox ini untuk menjalankan Linux mint tersebut tentunya dalam mode virtual alias simulasi. untuk lebih lengkapnya inilah fungsi fungsi virtualbox

#Fungsi

Oke sekarang lanjut ke funsinya.

#1. Mencoba Operation System apapun.
#2. Sebagai media untuk anda membuat simulasi jaringan.
#3. Sebagai komputer yang fleksibel dan dapat kita pindah-pindah sesuka hati kita.

# Berapa sih jumlah maksimal mesin virtual yang bisa di jalankan sekaligus pada Virtualbox ?

Sebanyak banyak nya , asalkan sumber daya PC atau laptop mampu. Satu mesin virtual yang aktif ada jatah RAM tersendiri, untungnya jumlah RAM yang digunakan dapat di setting, di kira kira saja berapa besar RAM yang dijatahkan agar mesin virtual tersebut tetap berjalan lancar. Jumlah banyaknya ram yang digunakan untuk mesin-mesin virtual akan mengurangi jumlah RAM yang digunakan untuk menjalankan OS utama. Amannya sih jangan sampai OS utama kekurangan sumber daya RAM. Selain itu kinerja prosesor juga akan terbagi.

# Apakah aman seandainya saya menggunakan mesin virtual secara terus menerus, untuk menggantikan komputer biasa gitu ?

Aman saja, bahkan saya menggunakan Virtualbox untuk menjalankan Server Database. Hidup 24 jam nonstop. Sebenarnya tergantung spesifikasi komputernya juga, kalau mpot mpotan menjalankan mesin virtual ya sebaiknya jangan.

# Kenapa tidak ada pilihan untuk membuat virtualisasi sistem operasi 64 bit pada virtualbox saya ?

Perlu di perhatikan, Jika ingin membuat dan menjalankan mesin virtual 64bit maka komputer utamanya harus ber OS 64 bit juga. sedangkan jika Komputer utama sobat sudah 64 bit maka seharusnya bisa menjalankan mesin virtual 32 maupun 64 bit. Namun pada kasus tertentu yang biasanya terjadi pada Windows 8 sekalipun host sobat sudah 64 bit tetap tidak bisa menjalankan virtual mesin 64 bit dikarenakan harus men setting terlebih dahulu pada BIOS dengan mengaktifkan fitur Virtualisasi di dalam menu security, Enable kan Intel Virtualization Tech dan fitur VT-d. lalu masuk control panel > turn windows feature on or off > matikan fitur Hyper-V.
